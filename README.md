## Setup 

Clone the project

```bash
  git clone https://github.com/memfisalinor/How-good-do-you-know-language-Json.git
```

Move to the folder

```bash
  cd How-good-do-you-know-language-Json
```

Use live server 
```bash
  npx live-server
```
